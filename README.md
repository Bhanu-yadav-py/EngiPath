# EngiPath
EngiPath is an AI-powered mobile app that provides engineering students personalized, exam-aware guidance, balancing academics, skill development, and placement preparation, while tracking readiness and offering milestone-based career roadmaps.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
